To build the operating system: 
	./buildos.sh

To run the operating system:
	./runos.sh

To test the operating system:
	Use the following commands:
		type <filename>
		execute <filename>
		copy <srcFile> <dstFile>
		dir
		create <filename>
		delete <filename>
		kill <pid>
		execfg <filename>
		quit
		help

	The following files are provided in the operating system:
		messag
		tstpr2
		phello
